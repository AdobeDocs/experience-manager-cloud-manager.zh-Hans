# 2020.9.0 版发行说明 {#release-notes-for}

## 发布日期 {#release-date}

版本2020.9.0 [!UICONTROL Cloud Manager] 的发布日期为2020年9月3日。

* 构建过程已分为三个单独的Maven命令。

* 在性能测试期间使用身份验证时，管道执行详细信息页面上现在有一个指示器。
