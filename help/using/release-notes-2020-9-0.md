# 2020.9.0 版发行说明 {#release-notes-for}

以下部分概述了[!UICONTROL Cloud Manager]版本2020.7.0的常规发行说明。

## 发布日期 {#release-date}

[!UICONTROL Cloud Manager]版本2020.9.0的发行日期是2020年9月3日。

## 新增功能 {#whats-new}

* 构建过程已分为三个单独的Maven命令。
* 如果git存储库克隆失败，则最多将重试三次该存储库。
* 现在，当在性能测试期间使用身份验证时，管道执行详细信息页面上会有一个指示器。
