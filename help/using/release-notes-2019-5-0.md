# 2019.5.0 版发行说明 {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.5.0版本不包含重大功能更改。 有关更多详细信息，请参阅以下各节。

## 发布日期 {#release-date}

[!UICONTROL Cloud Manager]版本2019.5.0的发布日期为2019年5月23日。


## 错误修复 {#bug-fixes}

* “SLA报告”图表在Y轴中显示重复值。
* “报表”屏幕对某些客户不可见。
* 在极少情况下，无法分析代码质量结果，导致流水线执行失败。
* 管线计划步骤中的日历未正确本地化。
* 各种小本地化问题和翻译改进。
