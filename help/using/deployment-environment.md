# 部署环境{#deployment-environment}

您可能是Adobe Experience Manager(AEM)6.4的新用户，或者需要升级到AEM 6.4版本。

如果您是AEM 6.4的新用户，那么您已经有权访问云管理器。

如果您是现有客户，则需要升级到AEM 6.4才能访问Cloud Manager。 从客户成功工程师(CSE)处收到URL和凭据后，即可使用Cloud Manager进行开始。
