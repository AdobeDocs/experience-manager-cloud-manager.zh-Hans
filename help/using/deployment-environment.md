# 部署环境{#deployment-environment}

您可能是Adobe Experience Manager(AEM)6.4的新用户或需要升级到AEM6.4版本。

如果您是AEM6.4的新用户，则您已经有权访问云管理器。

如果您是现有客户，则需要升级到AEM6.4才能访问云管理器。从客户成功工程师(CSE)收到URL和凭据后，您可以开始使用Cloud Manager。
