# 2020.10.0 版发行说明 {#release-notes-for}

以下部分概述了 [!UICONTROL Cloud Manager] 版本2020.10.0。

## 发布日期 {#release-date}

的发行日期 [!UICONTROL Cloud Manager] 版本2020.10.0为2020年10月1日。

## 错误修复 {#bug-fixes}

* 用于性能测试的爬网程序错误地将某些资源类型视为有效的Web链接。

* 在某些情况下，性能测试中的完成步骤处理不正确，从而导致步骤运行时间较长。

* 为生产部署配置调度程序缓存失效后，有时会执行两次失效。
