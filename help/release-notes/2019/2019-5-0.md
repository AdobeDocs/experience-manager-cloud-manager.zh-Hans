# Cloud Manager 2019.5.0发行说明 {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.5.0版本不包含重大功能更改。 有关更多详细信息，请参阅以下部分。

## 发布日期 {#release-date}

[!UICONTROL Cloud Manager] 2019.5.0的发布日期是2019年5月23日。


## 错误修复 {#bug-fixes}

* SLA报表图表在Y轴中显示重复的值。
* 某些客户看不到“报告”屏幕。
* 在极少数情况下，无法解析代码质量结果，从而导致管道执行失败。
* 管道计划步骤中的日历未正确本地化。
* 一系列次要本地化问题和翻译改进。
